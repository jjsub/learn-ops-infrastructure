# learn-ops-api: Service Exploration

## 1. Top-level folders in `learn-ops-api`

| Folder | Why does this folder need to exist? |
|--------|-------------------------------------|
| Config       | Holds server nginx config files                                    |
| LearningAPI  | Server and back-end, Controllers, Models, and DB business logic                                    |
| LearningPlatform    | Files to define the project logic like URL routes like Web Server Gateway Interface. and other midlewares                                    |
|Logs       |Keeping a log of the events of the app                                     |  
|LogViewer | third-party package or custom tool used to read, monitor, and manage the project's system log files directly through a web interface |
| static |This folder holds static files like css or images and other assets relevant for the application. |
|staticFiles |Static file that will be send to the browser to be loaded first. |
|templates |Holds the templates file ( HTML with mustaches) Where dynamic data will be injected into the template based of the business logic and DB values |
